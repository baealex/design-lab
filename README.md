# BaeJino Design

Conor / BaeJino의 공개 디자인 스케치북.

제품이 아니라, UI 시대의 인상과 작은 인터랙션을 모아둔 specimen gallery입니다.

A curated public sketchbook for UI-era specimens — trends, concepts, and effects.

## Taxonomy

- **Archive** — UI 시대와 시각 언어를 짧은 웹 표본으로 다시 보기
- **Concept** — 아직 제품이 아닌 분위기, 가설, 미래형 UI 스케치
- **Effect** — 작은 움직임과 인터랙션 실험

## Install

```bash
npm i
npm run dev
```

## Build

```bash
npm run build
```

## Note

각 페이지는 HTML 안에 SCSS와 TypeScript 문법을 함께 품고 있습니다.

VSCode에서 작업할 때 HTML validation이 거슬리면 관련 옵션을 꺼두는 편이 편합니다.

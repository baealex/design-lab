Inspired by Windows XP Luna

Luna A warmer visual language for Windows XP

Blue titlebar. Green Start button. Rolling hills.
A friendlier answer to gray desktop chrome.

📄
README.txt - Notepad

October 25, 2001. Windows XP launched with Luna — a visual style that replaced much of the previous gray, boxy chrome with blue title bars, a green Start button, and softly highlighted controls. The default "Bliss" wallpaper extended that blue-and-green palette from the windows to the desktop.

As PCs became everyday home and workplace tools, Luna made system structure feel more approachable through stronger color coding, larger shapes, and warmer imagery. Friendliness became part of the interface hierarchy.

The tradeoff was visual weight: oversized chrome and saturated color could compete with content. A modern reading keeps the clear grouping and welcoming tone, then reduces the amount of decoration around routine work.

Principle 01

Soft Highlights, Clear States

Stock Luna concentrated strong color in blue title bars and the green Start button; common controls were lighter and more restrained. The blue and green actions below deliberately abstract that system palette rather than reproduce a stock dialog button. Their soft satin highlight, border, and slight compression make state changes visible without borrowing Aqua's reflective gel effect.

Principle 02

Nature as Interface

Windows XP shipped with "Bliss" — a photograph of a green hill under a blue sky — as the default desktop. Its broad blue-over-green color fields worked with Luna's controls to create a consistent, optimistic environment rather than an unrelated backdrop. The cards below do not reproduce that photograph; they abstract its color structure into gradients so the composition can be studied without presenting the wallpaper as interface chrome.

Principle 03

The Friendly Surface

Earlier desktop chrome often used the same gray treatment for frames, controls, and content. Luna separated those roles with colored title bars, white work areas, rounded corners, and gentle shadows. The comparison below keeps that clearer grouping, then shows how a modern surface can reduce the decorative weight.

A Sunny Day Widget

Abstracted sky-to-meadow gradient + satin-highlighted controls + friendly white surfaces. Three principles, one interface. A weather widget that feels like opening a window. It uses Luna's color grouping and approachable tone in a bounded surface rather than spreading the full treatment across every workflow.

What This Is Not

⚠
Design Guidelines
  • ✗ Stock photography of happy people sitting on grass
  • ✗ Gradients without purpose or emotional direction
  • ✗ Corporate wallpaper masquerading as design
  • ✗ Glossy everything — some surfaces should breathe
  • ✓ Abstracted nature color fields as a bounded emotional backdrop
  • ✓ Softly highlighted controls with visible state changes
  • ✓ Warm, optimistic color temperature throughout
  • ✓ A welcoming tone that supports clear system structure

Keep the welcoming hierarchy.
Reduce the visual weight.

Color can distinguish structure without filling every surface.
Friendly does not have to mean oversized or ornamental.

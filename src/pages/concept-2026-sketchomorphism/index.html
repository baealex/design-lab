Original Concept

Sketchomorphism

Precision is easy to repeat.
Human variation is harder to fake.

Sketchomorphism explores deliberate variation in interfaces that otherwise feel mechanically uniform. Lines keep a hand-drawn edge, while each component type receives one stable outline from a small shared vocabulary. The goal is not to imitate paper everywhere. It is to leave a visible trace of making without sacrificing the structure underneath.

Rule 01

Lines With a Vocabulary

Borders are not randomized. Cards, controls, and notes each use an assigned outline with slightly different rhythm. A faint second stroke adds the feel of a pencil passing over the same edge twice.

Rule 02

Variation by Role

Variation is chosen, not generated. Components with the same role keep the same silhouette, so repetition creates familiarity instead of noise. Expression comes from the system of shapes, not from a new accident on every load.

Rule 03

Marks Respond to State

State changes the fill, mark, and text treatment without redrawing the component into a new shape. The outline stays familiar; the hand-made details make the result visible.

Limits

Keep the Structure Precise

  • ✗ Wobbly lines with no intent — imperfection without purpose
  • ✗ Illegible handwriting — readability is non-negotiable
  • ✗ Nostalgia cosplay — this is not a retreat to paper
  • ✓ Suits creative tools, personal products, education, and playful productivity
  • ✗ Avoid for dense tables, emergency controls, and high-frequency enterprise workflows
  • ✓ A small set of stable outlines assigned by component role
  • ✓ Precise layout and readable body type beneath the imperfect surface

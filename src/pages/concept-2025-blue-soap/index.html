Original Concept

Soap

Clean as blue soap. Soft as a bubble.
Light as foam.

Blue Soap is a one-hue interface study. It asks: what if the entire system had only one color?

One hue — #1194ff — scaled by opacity. Buttons at 100%. Shadows at 15%. Borders at 12%. Background tint at 3%. The constraint forces every choice to be deliberate. No color fights another because there's only one color to begin with.

The name comes from a bar of blue soap: simple, clean, and complete. The metaphor sets the mood, but the reusable idea is the constraint — one hue, many roles.

Rule 01

One Hue, Every Layer

rgba(17, 148, 255) — this is the only color. Buttons, shadows, borders, alerts, focus rings, background tint. All the same RGB, just different opacity. Even shadows are blue, not black. That's what makes it feel cohesive instead of assembled.

Rule 02

Let the Hue Pass Through

Surfaces stay close to white so the shared blue can tint borders, shadows, and the background without turning the page blue-on-blue. A small amount of transparency can let that tint pass through; blur is optional support, not the identity of the concept. Blue Soap is a color constraint, not a glass effect.

Rule 03

Only Actions Float

The blue shadow doubles as an affordance. Interactive elements lift slightly and spread that shadow; static surfaces do not move. Rounded forms support the soap metaphor, but motion is reserved for things a person can actually use. The controls below perform actions; the tags beneath them remain still.

Limits

When One Hue Is Enough

  • Suits focused utilities, settings, onboarding, and calm single-brand products
  • Avoid when many semantic categories must be distinguished by color
  • Blue text on everything — body text stays dark, blue is accent only
  • Black shadows on a blue palette — shadows match the hue
  • Blur on every card — transparency is optional, not the concept
  • Hover effects on static elements — only interactive things float
  • One hue at varying opacities builds every layer
  • Pale surfaces that preserve the shared hue
  • Float-up hover only on interactive elements
  • Blue-tinted shadows that belong to the palette
